About the System

Core Functions

  • Perform KEGG pathway enrichment analysis
  • Perform GO term enrichment analysis (BP, CC, MF)
  • Generate bar plots and dot plots for visualization
  • Export results to CSV files

Technical Background

  • Python backend with gseapy library
  • Uses Enrichr database for gene set enrichment
  • Supports multiple organisms (human, mouse, rat)
  • Interactive visualization with matplotlib

How to Use

  1. Enter gene list Input gene symbols (one per line) in the text area.
  2. Select gene sets Choose the gene set databases to analyze (KEGG and/or GO).
  3. Select organism Choose the target organism (human, mouse, or rat).
  4. Run analysis Click "Run Enrichment Analysis" to start the enrichment analysis.
  5. Explore results View the enrichment table and visualization plots.
